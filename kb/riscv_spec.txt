The RISC-V Instruction Set Manual
Volume I: Unprivileged Architecture

Integer Computational Instructions
Most integer computational instructions operate on XLEN bits of values held in the integer register file.
ADD: adds rs1 and rs2, writes to rd.
SUB: subtracts rs2 from rs1, writes to rd.
ADDI: adds sign-extended 12-bit immediate to rs1.
SLTI (set less than immediate) places the value 1 in register rd if register rs1 is less than the sign-extended immediate when both are treated as signed numbers, else 0 is written to rd.

Control Transfer Instructions
Branches: BEQ, BNE, BLT, BGE, BLTU, BGEU.
Branch instructions compare two registers. BEQ takes the branch if registers rs1 and rs2 are equal.
Jumps: JAL (jump and link), JALR. Jumps unconditionally and writes PC+4 to rd.

Load and Store Instructions
RV32I is a byte-addressable, little-endian architecture.
LW (load word) loads a 32-bit value from memory.
SW (store word) stores a 32-bit value to memory.
The effective address is obtained by adding the sign-extended 12-bit immediate to register rs1.
